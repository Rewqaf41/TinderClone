# Тестовое задание

**Реализация frontend части tinder.**

## Стек технологий

- React 18 + TypeScript
- Vite
- React Router
- Zustand
- React Query
- Tailwind CSS v4 + SCSS
- Framer Motion

## Установка и запуск

```bash
npm install
npm run dev
```

## Сборка

```bash
npm run build
npm run preview
```

## Линтинг

```bash
npm run lint
npm run format
```

## Структура проекта

Проект использует Feature-Sliced Design методологию.

## Git workflow

- `main` - продакшн ветка
- `develop` - разработка
- `feature/*` - ветки для фич
